# 🎯 Face Recognition System

> 📌 A Computer Vision project that detects and recognizes human faces using deep learning-based face encodings.

</div>

---

## 🚀 How to Run

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Face-Recognition-System.git
cd Face-Recognition-System
```

---

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Add your images

Put images inside the `database/` folder like this:

```
database/
   rohit_sharma/
      img1.jpg
      img2.jpg
   virat_kohli/
      img1.jpg
```

---

### 4️⃣ Run the program

```bash
python main.py
```

---

### 5️⃣ Enter image path

Example:

```
test.jpg
```
